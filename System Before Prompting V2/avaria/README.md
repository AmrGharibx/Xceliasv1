# Red Academy — Training Academy Management System

Red Academy is a premium, enterprise-grade dashboard for managing training batches, attendance, 10‑day summaries, and assessments. The UI is a modern dark theme with animated stats, real-time attendance feed, and interactive charts.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide Icons

## Getting Started

Install dependencies (already included in this repository) and run the dev server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Project Structure

- app/page.tsx — Red Academy dashboard UI
- app/layout.tsx — Fonts and metadata
- app/globals.css — Theme and global styles
